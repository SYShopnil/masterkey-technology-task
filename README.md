## Run Locally

Clone the project

```bash
  git clone https://github.com/SYShopnil/Maliha-poly-tex-fiber-industry-limited-task-client.git
```

Go to the project directory

```bash
  cd
  Maliha-poly-tex-fiber-industry-limited-task-client
```

Install dependencies

```bash
  npm install || npm i
```

Get .ENV file

```bash
  cp .env.example .env
```

Start the server

```bash
  npm run dev
```

Start the Test

```bash
  npm run test
```
