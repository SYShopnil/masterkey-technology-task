## Run Locally

Clone the project

```bash
  git clone https://github.com/SYShopnil/masterkey-technology-task.git
```

Go to the project directory

```bash
  cd
  masterkey-technology-task
```

Install dependencies

```bash
  npm install || npm i
```

Start the server

```bash
  npm run dev
```

Start the Test

```bash
  npm run test
```
